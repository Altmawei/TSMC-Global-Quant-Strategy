# TSMC Global Quant Strategy (V7.1)

這是一個專為其交所模擬競賽設計的量化交易系統，核心邏輯結合了全球市場連動、散戶心理分析與嚴格的動態風險控管。

## 🚀 核心功能
* **全球 ADR 溢價監控**：自動抓取美股 TSM 與台股 2330 價格，計算即時套利空間。
* **平盤震盪過濾器**：當日振幅低於 0.35% 自動封盤，保護本金避開無效洗盤。
* **結算日動態修正**：針對週三結算日的散戶踩踏慣性進行參數優化。
* **自動匯率對接**：整合 Yahoo Finance API 即時更新匯率，確保計算精確。

## 🛠️ 安裝與執行
1. 安裝必要套件：
   `pip install -r requirements.txt`
2. 執行策略腳本：
   `python main.py`
